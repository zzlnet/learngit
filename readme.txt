1、git init 初始化一个仓库

2、添加文件分两步
    git add <file> 可反复多次使用，添加多个文件
    git commit -m <message> 完成

3、
    git status      工作区状态
    git diff        查看修改内容

4、
    git log     显示从最近到最远的提交日志
    git log --pretty=oneline        显示简明信息
